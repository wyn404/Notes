预览：Ctrl + Shift + V 打开新窗口，加载md文件预览。

常用快捷键：

|  **key**         |  **command**    |
|      :----:     |   :----:|
| ctrl + b         | **粗体**       |
| ctrl + i         | *斜体*         |
| ctrl + shift + [ | 下一级标题     |
| ctrl + shift + ] | 上一级标题     |
| ctrl + M         | 不换行数学公式 |
| alt + S          | 删除线         |
